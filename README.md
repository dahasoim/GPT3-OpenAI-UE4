### checkRecognition function
1. 정해진 메뉴 리스트 가져온다.(UE 배열에 설정되어 있다.)
2. GPT3에서 처리된 text(=사용자 주문 메뉴)에서 불필요한 값을 정리한다.
3. 사용자가 주문한 메뉴가 기존 메뉴에 있는지 확인한다.
3.1 메뉴를 하나만 선택해야 하는 경우(단일 선택): 주문단계 1,2,7
3.2 주문단계 0:
